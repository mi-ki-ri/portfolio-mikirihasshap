バイオグラフィー / Biography

見切り発車Pは、広島県在住のボカロPです。

2010年に初音ミクを使用した楽曲を投稿しはじめ、それから年10曲～20曲程度のペースで活動を続けています。

音楽的には、フォーク/シンガーソングライター的な部分とネオソウル/ローファイ的な部分が共存しているのが面白いところだと思います。

様々なジャンルの音楽を貪欲に取り入れており、この先もさらに活動の幅を広げていきたいと考えています。
